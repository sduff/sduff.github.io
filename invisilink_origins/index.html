title: Invisilink Origins date: 2017-01-01 tags: [gaming] published: false layout: default series: gamedev --- # Invisilink Origins # Four In A Row # Memory feats of Chess and Go Masters Blindfolded One-Color Go https://senseis.xmp.net/?OneColourGo https://en.wikipedia.org/wiki/Go_variants#One_Color_Go # Historical It turns out, I wasn't the first to combine memory and four-in-a-row. In 1989, Parker Brothers released ["In The Dark"](https://boardgamegeek.com/boardgame/4133/dark), which involved a hidden four-in-a-row field. When one player believed they had won, the markers were revealed to show if they were correct or not. In 1992, a German publisher Piatnik released "4 Versteckt" which had the same premise. [BGG]
