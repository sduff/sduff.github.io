`WASTE` is a static site generator written in Python. *There are many like it, but
this one is mine*. I had a few ideas about how I wanted my site to work; many of
these ideas are implemented in `WASTE`, although other features have been dropped.

Features
--------

* Roughly 200 lines of Python
* Utilises [mistune](https://github.com/lepture/mistune)(Markdown) and [pygments](http://pygments.com)(code formatting)
* Tags are treated as standard pages, that can contain normal HTML, as well as backlinks
* Cool URIs by default
* Generates Sitemaps
* Per-Page templating
* Many metadata options
